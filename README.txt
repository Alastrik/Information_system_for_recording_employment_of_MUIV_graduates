# Информационная система учёта трудоустройства выпускников МУИВ

**Автор:** Монахов Артем Сергеевич  
**Тема ВКР:** Разработка информационной системы учета трудоустройства выпускников (на примере Частного образовательного учреждения высшего образования «Московский университет имени С.Ю. Витте»)

## Описание
Десктопное приложение на Python (Tkinter) для учёта и анализа трудоустройства выпускников МУИВ.  
Поддерживает 3 роли пользователей, генерацию отчётов, личный кабинет, панель администратора.

## Учётные записи для проверки

| Роль          | Логин            | Пароль        |
|---------------|------------------|---------------|
| Администратор | `admin`          | `admin123`    |
| HR-специалист | `hr_muiiv`       | `hrpass2025`  |
| Выпускник     | `graduate_001`   | `grad2025`    |

> ⚠️ Пароли хранятся в открытом виде **только для учебной практики**.

## Требования
- Python 3.9+
- PostgreSQL 12+

## Установка
1. Убедитесь, что PostgreSQL запущен и существует БД `muiv_employment`.
2. Выполните `pip install -r requirements.txt`
3. Запустите `python main.py`

## Сборка в .exe
```bash
pyinstaller --onefile --windowed --add-data "resources;resources" main.py
